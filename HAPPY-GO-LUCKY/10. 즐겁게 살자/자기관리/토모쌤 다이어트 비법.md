다이어트 비법 물어보시는 분이 많아서 공유합니다.

한달 플랜이에요. 전 81kg에서 69kg으로. 허리 34인치에서 28인치로 마무리.

- 전 기간 금지 : 술, 백미, 밀가루, 지방, 양념, 그리고 사람.
- 전 기간 필수 : 고급 프로바이오틱스(유산균) 복용(쎌티아이 등). 많은 물, 외로울 것(사람 만나지 말고). 숙면(지방은 잘 때 소변으로 빠짐)

ㅇ 1~3일차(늘어난 위장 줄이기)

- 식단 : 일반 식사 중단. 유청 단백질파우더 하루 4끼, 허기지면 녹황색 채소나 오이 섭취, 영양제 섭취(종합, 코큐텐, 오메가, 칼슘/아연/마그네슘, 추가 비타민C 등. 생략 가능)
- 운동 : 금지. 운동까지 할 힘이 없음.
- 기대 : 2~3kg 체중 감소하나 대부분 수분이니 지레 기뻐하지 말 것
- 조언 : 매우 허기질 수 있으나, 3일만 딱 참을 것.

ㅇ4일~7일(워밍업)

- 식단: 현미밥 시작. ONLY 현미밥 + 녹채소 쌈 하루 3끼. 쌈장 허용.
- 운동 : 가벼운 운동 시작. 30분이상 걷기에 준하는 운동량.
- 기대 : 별 성과 없는 기간. 여기까지 1주일 견뎌내면 절반은 성공.

ㅇ 8일~15일(본격 다이어트)

- 식단 : 기존 식단 유지. 가끔 지방 적은 고기정도 허용.
- 운동 : 짧고 강한 운동 추천(고강도 인터벌 운동 등). 30분이상 헐떡거려야 함.
- 기대 : 매일 300g 이상 감량
- 조언 : 별로 허기지지 않으니 유혹에만 넘어가지 말 것.

ㅇ 16일~30일(피똥 싸는 기간)

- 식단 : 전과 동
- 운동 : 근력 운동 추가. 무조건 헬스장가서 근육 붙일 것.
- 기대 : 인생이 달라짐~
